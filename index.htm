Lightweight Java program for easily downloading subtitles for your video files

Get perfectly matching subtitles from OpenSubtitles.org

Instead of having to go to a website and search for subtitles, downloading them, putting them in the correct folder and naming them correctly, Casub does all this for you.

Simply choose a video file

And Casub will fetch the subtitles for you in an instant.

Done!

Watch your movie with perfectly matched subtitles!

How to use

Allow the program!

For now this is only for people I know, so this is how it works. If you download the program it will block you from using it, since it could be a virus etc. If you trust me though, follow these simple steps.

Obviously you'll need Java to run my Java program. Then you need to tell Java that you trust programs downloaded from this website. Download Java HERE.


For Mac

1. Open System Preferences -> Java -> Security
2. Edit website list -> Add http://repsacc.github.io

For Windows

1. Click on the Start menu -> Search for Java Control Panel
2. Click the Start menu -> Programs -> Java -> Configure Java
3. Go to Security -> Edit website list
4. Add http://repsacc.github.io

Last step is to download the program, which you can do HERE.
