A lightweight program for easily downloading subtitles for your video files

Subtitles from OpenSubtitles.org

Casub does all the work for you. It fetches perfectly matching subtitles for your video file - given that such are supposed to exist!

Simply choose a video file

And Casub will fetch the subtitles for you in an instant.

The result

The subtitles are added.

Done!

The subtitle file is correctly named and placed in the same folder.

How to use

Download Java!

You will need Java to run my Java program. If you do not know if you have Java, go download it (which will update it for you).
