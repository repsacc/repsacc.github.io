How to use

If you download the program it will block you from using it, since it could be a virus etc. If you trust me though, follow these simple steps:

  • Obviously you'll need Java to run my Java program. Do that here
  • Then you need to tell Java that you trust programs downloaded from this website.
    • For Mac:
    1. Open System Preferences -> Java -> Security -> Edit website list.
    2. Add http://repsacc.github.io
    For Windows:
    1. Click on the Start menu -> Search for Java Control Panel
    2. Click the Start menu -> Programs -> Java -> Configure Java
    3. Go to Security -> Edit website list.
    4. Add http://repsacc.github.io
